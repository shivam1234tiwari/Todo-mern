# 📝 MERN Todo App (Dockerized)

A full-stack **Todo Application** built using the MERN stack (MongoDB, Express, React, Node.js) and containerized using Docker.

---

## 🚀 Features

- ✅ Add, Edit, Delete Todos
- ✅ REST API with Express & Node.js
- ✅ MongoDB Database Integration
- ✅ React Frontend (Vite)
- ✅ Dockerized (Frontend + Backend + Database)
- ✅ Environment Variable Support (.env)
- ✅ Scalable Project Structure

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Containerization: Docker & Docker Compose
